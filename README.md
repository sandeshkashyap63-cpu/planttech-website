# Planttech Engineering — Homepage

Static homepage for **Planttech Engineering Pvt. Ltd.** (Laksar, Haridwar) —
manufacturer of process machinery for herbal, nutraceutical, pharmaceutical
and chemical plants.

## Files
- `index.html` — the complete homepage (HTML + CSS + JS in a single file)

## Contact form
The "Request a Quote" form submits via [Web3Forms](https://web3forms.com/).
To receive submissions in your inbox, replace `YOUR_WEB3FORMS_KEY` in
`index.html` with your own free Web3Forms access key.

## Local preview
Open `index.html` directly in a browser, or serve the folder:

    python3 -m http.server 8000

then visit http://localhost:8000
